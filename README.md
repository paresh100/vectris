# Master Antigravity Template

A premium Next.js 15+ portfolio template designed for high-end digital craftsmanship.

## Features
- **Smooth Scrolling** (Lenis)
- **Page Transitions** (Framer Motion)
- **SEO Optimized** (JSON-LD, Dynamic OpenGraph, Sitemap)
- **Programmatic Video** (Remotion)
- **Marketing Skills** (AI Agent Integration)

## Setup
1. Clone the repository.
2. Run `npm install`.
3. Run `npm run setup` to configure your project key details.
4. Run `npm run dev` to start.
